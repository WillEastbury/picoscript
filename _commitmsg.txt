GPIO reference emulator runtime (Gpio.* on Python + JS VMs)

Makes Gpio.* functional in the reference simulator/debugger (the hooks shipped as
ABI in ac2b1fe; the runtime was NOOP). HostApi._gpio (picoscript_vm.py) + the
vm/picovm.js _gpio mirror, byte-identical: per-pin {dir,pull,value}; value
saturates to [0,1024]; dir 0=in/1=out; pull 0=none/1=up/2=down; Count()=40
(reference header size). GPIO is platform-injected (PIOS supplies the real per-pin
driver), so the C VM stays NOOP and behaviour is asserted as Python VM == JS VM
rather than on the 5-path gate. tests/test_devices.py. Regenerated playground.html.
