PicoScript is a deterministic bytecode language for userland message processing in the PIOS kernel. Each method compiles to a single 32-bit instruction (OP_NOOP + hook code).

Hook Code Allocation

RangeCategoryMethodsPurpose
0x01-0x70Core (Hardware)44Kernel, Memory, Threading
0x80-0xC6Standard Library63String, Number, Math
0xD0-0xFEContext/System39Environment, Crypto
0x0100-0x0149Application Features44Http, Auth, Html, X509

Language Syntax & Keywords

FOR

FOR (init; condition; increment) { statements }
Description:
C-style loop with init, condition, and increment.
Example:
FOR (i = 0; i < 100; i++) { Memory.Poke(addr + i, 1, 0); }

FOREACH

FOREACH (item IN collection) { statements }
Description:
Iterate over elements in a collection or descriptor array.
Example:
FOREACH (header IN headers) { process_header(header); }

GOSUB

GOSUB subroutine_name;
Description:
Call a subroutine; returns to next instruction after call.
Example:
GOSUB validate_request;

GOTO

GOTO label_name;
Description:
Jump to a labeled location in code.
Example:
GOTO error_handler;

IF

IF (condition) { statements }
Description:
Conditional execution. Executes the block if condition evaluates to true.
Example:
IF (x > 5) { Memory.Poke(0x1000, 1, 42); }

RETURN

RETURN [value];
Description:
Exit function/subroutine with optional return value.
Example:
RETURN Http.GenerateResponse(200, headers, body);

SWITCH

SWITCH (expr) { CASE val: statements; DEFAULT: statements; }
Description:
Multi-way branching based on expression value.
Example:
SWITCH (status) { CASE 200: handle_ok; CASE 404: handle_not_found; }

WHILE

WHILE (condition) { statements }
Description:
Loop that executes while condition is true.
Example:
WHILE (remaining > 0) { chunk = read_next(); remaining--; }
